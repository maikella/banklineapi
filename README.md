Nesse projeto, fiz uma aplicação que gera **Extratos Bancários**!

### Principais tecnologias utilizadas:

 - Java e Spring Boot

### Lições aprendidas:

- Modelar e desenvolver uma API REST
- Criar documentação com o Swagger
- Acessar um banco de dados (Postgres)
